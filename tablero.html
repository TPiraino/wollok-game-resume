🎯 Tablero y Posiciones

Domina el sistema de coordenadas y configuración del tablero

🏗️ El Tablero

El tablero es el corazón de tu juego. Es donde se ubican todos los objetos visuales y tiene características específicas:

📋 Características del Tablero

  • Único: Solo puede haber un tablero por juego
  • Dividido en celdas cuadradas: Configurables con game.cellSize(px)
  • Posiciones en unidades de celdas: No se puede estar "en el medio"
  • Configurable: Título, ancho, alto e imagen de fondo

⚙️ Configuración del Tablero

Propiedades Básicas

import wollok.game.*

program configuracionTablero {
    game.width(10)           // Ancho en celdas
    game.height(7)           // Alto en celdas  
    game.cellSize(50)        // Tamaño de celda en píxeles (default: 50x50)
    game.title("Mi Juego")   // Título de la ventana
    game.ground("fondo.png") // Imagen de fondo para las celdas
    
    game.start()
}

📏 Propiedades Disponibles

  • game.width(celdas) - Ancho del tablero
  • game.height(celdas) - Alto del tablero
  • game.cellSize(píxeles) - Tamaño de cada celda
  • game.title(texto) - Título de la ventana
  • game.ground(imagen) - Imagen de fondo de las celdas

📍 Sistema de Posiciones

Las posiciones definen coordenadas x e y en el tablero. Son objetos que representan un punto específico.

Crear Posiciones

// Crear una posición específica
const posicion = game.at(2, 3)  // x = 2, y = 3

// Posiciones especiales
const origen = game.origin()     // 0@0
const centro = game.center()     // Se calcula automáticamente

// Acceder a coordenadas
posicion.x()  // 2
posicion.y()  // 3

🎯 Posiciones Especiales

  • game.origin() - Esquina superior izquierda (0@0)
  • game.center() - Centro del tablero (calculado automáticamente)
  • game.at(x, y) - Posición específica en coordenadas x, y

🎨 Configurar Imagen de Fondo

Puedes personalizar el aspecto de las celdas del tablero:

import wollok.game.*

program tableroConFondo {
    game.width(8)
    game.height(6)
    game.cellSize(60)
    game.title("Tablero con Fondo")
    game.ground("cesped.png")  // Imagen de fondo para cada celda
    
    game.start()
}

🖼️ Tipos de Imágenes de Fondo

  • Cesped: Para juegos de aventura
  • Piedra: Para mazmorras
  • Agua: Para niveles acuáticos
  • Nieve: Para niveles invernales

📐 Ejemplo Completo

Un ejemplo completo de configuración de tablero:

import wollok.game.*

program ejemploCompleto {
    // Configuración del tablero
    game.width(12)
    game.height(8)
    game.cellSize(40)
    game.title("Mi Primer Juego")
    game.ground("tierra.png")
    
    // Crear algunas posiciones de ejemplo
    const esquina = game.origin()
    const centro = game.center()
    const posicionEspecial = game.at(5, 3)
    
    // Mostrar información en consola
    console.println("Esquina: " + esquina)
    console.println("Centro: " + centro)
    console.println("Posición especial: " + posicionEspecial)
    
    game.start()
}

💡 Consejos Importantes

  • Planifica el tamaño: Piensa en cuántos objetos necesitas antes de configurar el tablero
  • Celdas cuadradas: El tablero siempre usa celdas cuadradas, no rectangulares
  • Posiciones enteras: No puedes colocar objetos entre celdas
  • Imágenes pequeñas: Las imágenes de fondo se repiten en cada celda
