¡Bienvenido a la Guía de Wollok Game!

Esta guía te ayudará a crear juegos increíbles usando la biblioteca Wollok Game. Está diseñada especialmente para estudiantes universitarios.

🚀 Inicio Rápido

import wollok.game.*

program miJuego {
    game.width(10)
    game.height(7)
    game.title("Mi Primer Juego")
    game.start()
}
